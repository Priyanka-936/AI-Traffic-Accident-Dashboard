# AI-Powered Traffic Accident Analysis Dashboard

An AI-based dashboard that identifies accident hotspots, analyzes accident causes, and predicts accident severity using Machine Learning.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Flask
- React
- Tailwind CSS

## Features

- Accident Hotspot Detection
- Severity Prediction
- Weather Analysis
- Risk Zone Mapping
- Interactive Dashboard
